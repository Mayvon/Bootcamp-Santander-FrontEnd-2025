Introdução ao HTML na Prática

HTML (HyperText Markup Language) é a linguagem que estrutura o conteúdo de páginas web. Um documento HTML é formado por elementos (tags) que definem títulos, parágrafos, links, imagens e seções da página.

Estrutura mínima:

<!DOCTYPE html>
<html lang="pt-br">
  <head> <meta charset="UTF-8"> <title>Título</title> </head>
  <body>
    <h1>Olá, mundo!</h1>
    <p>Este é um parágrafo.</p>
  </body>
</html>

Tags comuns: h1–h6, p, a, img, ul/ol/li, div, span.

Estruturando seu HTML + Formatações

Estruturar bem o HTML facilita leitura, manutenção e acessibilidade. Use títulos hierárquicos, listas para itens, e formatação textual apenas quando necessário.

Exemplos de formatação: strong (destaque), em (ênfase), mark (marcar), code (código), br (quebra de linha).

<h2>Seção</h2>
<p>Texto com <strong>destaque</strong> e <em>ênfase</em>.</p>
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Boas práticas: evitar usar formatação só visual (ex.: várias <br> para espaçar); preferir organização lógica com seções e listas.

Trabalhando com Formulários em HTML

Formulários coletam dados do usuário. Cada controle possui um name que identifica o valor enviado.

<form action="/enviar" method="post">
  <label for="nome">Nome:</label>
  <input id="nome" name="nome" type="text" required>

  <label for="email">E-mail:</label>
  <input id="email" name="email" type="email" required>

  <label for="msg">Mensagem:</label>
  <textarea id="msg" name="mensagem"></textarea>

  <button type="submit">Enviar</button>
</form>

Campos úteis: text, email, number, date, checkbox, radio, file, select/option. Atributos como required, min, max e placeholder melhoram a experiência.

Trabalhando com Mídias utilizando HTML

Imagens, áudios e vídeos enriquecem o conteúdo. Sempre descreva imagens com alt para acessibilidade.

<img src="foto.jpg" alt="Descrição da imagem">

<audio controls>
  <source src="trilha.mp3" type="audio/mpeg">
  Seu navegador não suporta áudio.
</audio>

<video controls width="400">
  <source src="video.mp4" type="video/mp4">
  Seu navegador não suporta vídeo.
</video>

Dica: forneça múltiplos formatos quando possível (ex.: .mp4 e .webm), e use textos alternativos.

Criando Tabelas com HTML

Use tabelas para dados tabulares (linhas e colunas). Não use tabelas para layout.

<table>
  <caption>Vendas por Mês</caption>
  <thead>
    <tr><th>Mês</th><th>Total (R$)</th></tr>
  </thead>
  <tbody>
    <tr><td>Jan</td><td>10.000</td></tr>
    <tr><td>Fev</td><td>12.500</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Total</td><td>22.500</td></tr>
  </tfoot>
</table>

Elementos-chave: table, caption, thead, tbody, tfoot, tr, th, td.

Entendendo HTML Semântico

HTML semântico usa tags que descrevem o papel do conteúdo, melhorando SEO, acessibilidade e manutenção.

<header>Cabeçalho da página</header>
<nav>Links de navegação</nav>
<main>
  <article>
    <h1>Título do artigo</h1>
    <section>Um tópico do artigo</section>
    <aside>Nota lateral</aside>
  </article>
</main>
<footer>Rodapé</footer>

Quando usar: header (topo), nav (menus), main (conteúdo principal), section (seções temáticas), article (conteúdo independente), aside (complementos) e footer (rodapé).
